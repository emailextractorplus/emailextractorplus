Free Email Grabber is a simple, easy-to-use program that can extract email addresses from websites, search engines, local files, and other data sources.

Free Email Grabber lets you limit the range of scanning in a number of ways, so that you can extract only specific addresses from webpages. You can restrict the scanning range, use filters to skip unnecessary links (or to remove those links during the extraction process). The program can extract email addresses from static sites, script-based forums, blogs, or any CMS-based sites. It can extract not only addresses but their owners’ names too, so you can make higher-quality mailing lists.

When searching by keywords, you can use more than 100 popular search engines, including Google, Yahoo, MSN, Altavista, and so on.

Free Email Grabber is based on the Internet Explorer engine, so it’s highly compatible with all web servers.

The program features:

  • A great choice of search engines (over 100), with the option to manually tune the search templates;

  • A built-in parser;

  • A flexible system for restricting the scanning range;

  • Command line support.
