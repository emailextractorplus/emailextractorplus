Installation of the Email Grabber:

Email Grabber is distributed as an .EXE-file. To install EE, launch "EmailExtractorSetup.exe" then follow the Installation Wizard instructions. After successful install you can use the Email Grabber. Removal of the program:

To remove the Email Extractor from your system:
1. Open Control Panel window.
2. Double-click on "Add or Remove Programs" icon
3. Find the program name in installed applications list, select it and click on "Add/Remove" button.
4. Follow Uninstall Wizard instructions.

OR:

Select Start/Programs/Email Grabber/Uninstall and follow Uninstall Wizard instructions.
